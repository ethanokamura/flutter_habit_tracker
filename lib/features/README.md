# Features:
Implementations of the packages directory (The UI).

```
lib/features/
    ├── authentication/           # Handles user login and registration pages
    ├── failures/                 # Handles cubit failures for the habits and users
    ├── habits/                   # Handles everything needed to implement the habits!
    ├── heatmap/                  # Handles the custom heatmap display
    ├── positive_rabit/           # Handles the habit rabbit's daily message
    ├── profile/                  # Handles user features and app settings
    └── unknown/                  # Handles unexpected pages
```
